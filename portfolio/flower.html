2021/09 Androidアプリ / Java

植物育成アプリの作成

editor : AndroidStudio
言語 : Java
端末 : Lenovo Tablet TB-X104F
制作期間：1週間

初期画面

種をタップすると芽が出ます


太陽かマーライオンをタップすると下部に「太陽を浴びました」もしくは「水をあげました」と表示し２秒後に消えます
プログラムで回数をカウントし一定の条件を満たすと芽が成長します


バランスよくお世話すると花が咲きます


太陽にあてすぎるとかれます


水をあげすぎると枯れます


満開の花か枯れた花をタップすると最初に戻ります


途中で植物をタップすると「太陽か水をあげよう」と表示します


工夫したこと

ImageButtonを使用して太陽やマーライオンをボタンにしました
タップした際に押した感が出るように画面に触れているときは縮小するようにプログラミングしました


改善点

遊んでみるとパターンや動きが少ないためすぐに飽きてしまうと感じました
アニメーションをつけるなどできるとよかったと思います
